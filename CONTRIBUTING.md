# Contributing to Naija Dev Toolkit

Thank you for wanting to contribute! 

This project is open-source, and we welcome contributions of any kind: code, translations, bug fixes, design improvements, or ideas.

---

## ðŸ›  How to Contribute

1. **Fork the repo**  
   Click the fork button at the top right to create your copy.

2. **Clone your fork locally**
```bash
git clone https://github.com/YOUR-USERNAME/naija-dev-toolkit.git
```

3. **Create a new branch**
 - git checkout -b feature/your-feature-name
 - Make your changes
 - Follow existing code style
 - Test all features locally (English & Pidgin modes!)
 - Commit your changes

4. ***Commit your changes***
 - git add .
 - git commit -m "Add: description of your feature/fix"
 - Push to your fork
 - git push origin feature/your-feature-name

5. ***Open a Pull Request***
 - Explain your changes clearly
 - Reference any related issues if applicable

***Good First Issues***

 - Add more Nigerian banks to the bank lookup
 - Add USSD codes for popular services
 - Improve translations for Pidgin mode
 - Improve mobile responsiveness of cards and inputs
 - Keyboard support for bank dropdown (Arrow keys, Enter, Esc)
 - Accessibility improvements (labels, contrast, ARIA roles)

  ***Guidelines***

 - Keep code clean and readable

 - Test in both English & Pidgin

 - Only commit relevant changes

 - Respect the existing project style
Thank You

Your contribution helps Nigerian developers save time and makes the toolkit even better.
Every PR counts even small tweaks!
